# Rose Bud Thorn – React UI

## Run

1. Start the Java backend: from project root run `mvn spring-boot:run` (port 8080).
2. From this folder run:
   ```bash
   npm install
   npm run dev
   ```
3. Open http://localhost:5173 in your browser.

The dev server proxies `/api` to the backend, so no login is required; entries use a default user (id 1).
